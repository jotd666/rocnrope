"# Roc'NRope" 
port of Roc'NRope for Amiga

Requires a 2-button joystick, or the keyboard

Features:

- 50 fps on unexpanded A1200 (using dual playfield 16+16 colors + hardware sprites)
- 25 fps on unaccelerated A500 1MB (using 16 colors + hardware sprites)

Credits:

  sound conversion for the Amiga.

Instructions:

fire/5: insert coin
up/down/1/2: start game
joystick/arrows: move
red/ctrl: shoot grappling hook
blue/alt: dazzle
ESC: quit current game

Cheat keys:

F1: skip level
F3: add 10000 points
F4: toggle invincibility
