"# Roc'NRope" 
port of Roc'NRope for Amiga

Requires a 2-button joystick, or the keyboard

Credits:

- jotd: reverse-engineering, 68000 transcode, graphics conversion
  sound conversion for the Amiga.
- no9 (soon): music
- PascalDe73: icons
- Konami: original game

Instructions:

fire/5: insert coin
up/down/1/2: start game
joystick/arrows: move
red/ctrl: shoot grappling hook
blue/alt: dazze

Cheat keys:

F1: skip level
F3: add 10000 points
F4: toggle invincibility
